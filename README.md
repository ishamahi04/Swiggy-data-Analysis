# 🍽️ Swiggy Data Analysis

## 📌 Project Overview
This project analyzes Swiggy restaurant dataset using *Python (Jupyter Notebook)* and *Power BI* to extract meaningful insights about restaurant trends, ratings, pricing, locations, and delivery patterns.

The goal of this project is to perform *Exploratory Data Analysis (EDA)* and build *interactive dashboards* for better understanding of food delivery data.

## 📊 Dataset Information

The dataset contains restaurant-related information such as:

- Restaurant Name
- Category
- Rating
- Cost for Two
- Veg / Non-Veg
- City
- Area
- Locality
- Address
- Long Distance Delivery

## 🎯 Objectives

- Clean and preprocess raw restaurant data
- Perform Exploratory Data Analysis (EDA)
